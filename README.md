# Construction Scheduler

A beginner Java application for managing construction projects and scheduled tasks.

## Planned Features

- Create construction projects
- Add tasks to projects
- View project schedules
- Update project statuses
- Update task statuses
- Validate project and task information

## Technologies

- Java
- IntelliJ IDEA
- Git
- GitHub

## Future Improvements

- File storage
- JUnit testing
- Spring Boot
- REST API
- Database storage
